ROLLBACK
================================================================

Si l'app est instable apres deploiement:

PHASE 1 — Gunicorn (variables Coolify):
  GUNICORN_WORKERS=2
  GUNICORN_TIMEOUT=600

PHASE 2 — Tokens (si threading cause des problemes):
  Les jobs de refresh sont idempotents, relancer manuellement si besoin

PHASE 3 — Cache assets:
  Si les assets sont caches trop longtemps, forcer un hard refresh (Ctrl+Shift+R)
  ou changer le mtime d'un fichier pour invalider le cache
