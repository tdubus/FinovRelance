Configuration du Connecteur Excel/CSV

Comment configurer votre import

Le Connecteur Excel/CSV vous permet d'importer des clients et factures depuis des fichiers Excel (.xlsx) ou CSV (.csv) sans reformater vos fichiers à chaque fois.

  1. Uploadez un fichier exemple (Excel ou CSV) pour détecter automatiquement les colonnes
  2. Associez chaque colonne de votre fichier aux champs FinovRelance
  3. Sauvegardez la configuration (elle sera réutilisée pour tous vos futurs imports)
  4. Importez vos clients et factures directement depuis les Paramètres
