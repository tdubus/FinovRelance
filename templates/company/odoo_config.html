Comment obtenir votre clé API Odoo ?
  1. Connectez-vous à votre instance Odoo
  2. Cliquez sur votre nom en haut à droite
  3. Sélectionnez "Mon profil"
  4. Allez dans l'onglet "Sécurité du compte"
  5. Cliquez sur "Nouvelle clé API"
  6. Donnez un nom à la clé (ex: "Finov Relance")
  7. Copiez la clé générée et collez-la ci-dessus
Note : Les clés API Odoo n'expirent pas automatiquement. Conservez-la en lieu sûr !
